# 🤖 AI Agent System (Tool-Based Chat Assistant)

## 📌 Overview
This project is a simple AI agent built in Python.  
It simulates a chatbot that can understand user input and use different tools like calculator, time, and discount system.

The goal of this project is to demonstrate how an **AI agent decides actions based on user input**.

---

## 🧠 How It Works

User → Agent (brain/decision layer) → Tools (actions)

- agent.py → decides what to do
- tools.py → performs actions
- main.py → chat interface

---

## ⚙️ Features

- 💬 Chat-style interface
- 🧠 Rule-based AI agent
- 🧮 Calculator tool
- 🕒 Time display tool
- 🛒 Discount redemption system
- 🧪 Pytest test cases
